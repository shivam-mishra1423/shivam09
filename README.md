# Shivam Kumar Mishra — Portfolio

🚀 Ultra-modern portfolio built with React (CDN) + Vanilla JS

## Features
- Custom animated cursor
- Scroll progress bar
- React Typewriter effect in Hero
- React Skill Proficiency bars with animation
- Smooth reveal animations on scroll
- Mobile responsive with hamburger menu
- Contact form with success feedback
- Dark space-themed aesthetic

## 📁 Files
```
shivam-portfolio/
├── index.html      ← Main portfolio (all in one)
├── vercel.json     ← Vercel deployment config
├── shivam.pdf      ← Add your resume here
└── README.md
```

## 🚀 Deploy to Vercel (2 minutes)

### Method 1: GitHub (Recommended)
1. Create a new GitHub repo
2. Upload these files to the repo
3. Go to [vercel.com](https://vercel.com) → New Project
4. Connect your GitHub repo
5. Framework: **"Other"** → Deploy ✅

### Method 2: Vercel CLI
```bash
npm i -g vercel
cd shivam-portfolio
vercel
```

### Method 3: Drag & Drop
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag the `shivam-portfolio` folder directly
3. Deploy! ✅

## 📝 Customization
- Add your **photo**: Replace the initials block with `<img src="shivam.jpg">`
- Add your **resume**: Put `shivam.pdf` in the same folder
- Update **GitHub/LinkedIn links** in the social section
- Update **project GitHub links** on proj-card elements
